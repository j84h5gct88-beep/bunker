# BUNKER Lounge Bar

Новая коммерческая версия сайта с классической структурой премиального lounge-проекта.

## Структура
- Первый экран и CTA
- О заведении и показатели
- Форматы отдыха
- Преимущества
- Галерея
- VIP-зона
- Афиша
- Отзывы
- Контакты и карта
- Анкета бронирования через WhatsApp

## Перед публикацией
1. Заменить демонстрационные изображения на реальные фотографии BUNKER.
2. Подтвердить номер WhatsApp +7 919 113-09-32.
3. Добавить реальные отзывы с разрешения авторов.
4. Проверить точную ссылку на карточку 2ГИС.

Для локального запуска:
python -m http.server 8080
